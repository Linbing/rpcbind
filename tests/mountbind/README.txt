Порядок работы:

network-config.sh - конфигурирует простую схему с двумя namespaces
network-check.sh - проверяет работоспособность схемы(с помощью ping)

Дополнительно:
clear_ns - удаляет все имеющиеся namespaces
clear_dev - удаляет сетевые интерфейсы(можно выбрать один или несколько из предложенного списка)

server - запускает rpcserver
client [server_address] "Your mesage" - запускает клиента, который отправляет на сервер сообщение, записывает его в файл и получает ответ об успешном/неуспешном завершении
